# Render部署视频脚本

## 视频信息

- **标题**：3分钟部署批量证书生成器到公网
- **时长**：3-5分钟
- **目标**：教会观众免费部署Web应用到公网

---

## 开场（30秒）

**画面**：显示智能文档工厂界面
**配音**：
> "大家好，今天教大家如何把这个批量证书生成工具部署到公网，让任何人都能使用。全程免费，只需要3分钟。"

**字幕**：
- 批量证书生成器
- 免费部署到公网
- 3分钟完成

---

## 第一步：注册Render（45秒）

**画面**：
1. 浏览器打开 render.com
2. 点击 "Get Started for Free"
3. 点击 "Continue with GitHub"

**配音**：
> "首先访问render.com，点击免费开始，用GitHub账号登录，不需要额外注册。"

**字幕**：
- 访问 render.com
- 点击 Get Started
- 用GitHub登录

**画面**：
4. GitHub授权页面
5. 点击 "Authorize render"
6. 进入Render Dashboard

**配音**：
> "授权Render访问你的GitHub，这样就完成了注册。"

---

## 第二步：创建服务（60秒）

**画面**：
1. 点击 "New +"
2. 选择 "Web Service"

**配音**：
> "登录后，点击New，选择Web Service，我们要创建一个网站服务。"

**画面**：
3. 搜索 doc_factory
4. 点击 Connect

**配音**：
> "找到我们的代码仓库，点击连接。"

**画面**：
5. 配置页面
   - Name: doc-factory
   - Region: Singapore
   - Runtime: Python 3

**配音**：
> "填写服务名称，选择新加坡节点，这样国内访问更快。运行时选择Python 3。"

**画面**：
6. Build Command: pip install -r requirements.txt
7. Start Command: python app_v2.py

**配音**：
> "构建命令安装依赖，启动命令运行我们的程序。"

**画面**：
8. Instance Type: Free

**配音**：
> "选择免费套餐，足够我们使用了。"

---

## 第三步：等待部署（45秒）

**画面**：
1. 点击 "Create Web Service"
2. 显示部署日志
3. 日志滚动：Installing dependencies...

**配音**：
> "点击创建，等待部署完成。可以看到实时日志，正在安装依赖。"

**画面**：
4. 日志显示 "Your service is live"
5. 页面显示绿色 "Live"

**配音**：
> "当看到服务上线，说明部署成功了！"

---

## 第四步：测试使用（60秒）

**画面**：
1. 点击分配的URL
2. 打开智能文档工厂界面

**配音**：
> "点击这个链接，就是我们的公网地址。任何人都能访问。"

**画面**：
3. 选择荣誉证书模板
4. 输入姓名：张三、李四
5. 填写活动名称
6. 点击生成证书
7. 下载zip文件

**配音**：
> "测试一下功能，选择模板，输入姓名，点击生成，可以正常下载证书。完美！"

**画面**：
8. 解压zip，显示生成的证书文件

**配音**：
> "解压看看，证书都正确生成了。"

---
